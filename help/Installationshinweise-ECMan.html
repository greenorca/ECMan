Binärdistribution

Die Binärdistribution ist ein künstliches Kompilat mit (fast) allen internen Abhängigkeiten. Es existiert zunächst nur für Windows 10.

Erforderliche Pakete

Auf Windows-Rechnern wird für die Erstellung der PDF-Logdateien wkhtml2pdf benötigt. Ferner muss das Visual C++ 2015 (x86) Paket installiert werden: https://www.microsoft.com/de-de/download/confirmation.aspx?id=53840

Source-Distribution

ECMan ist mit Python programmiert. Dementsprechend kann der Quellcode in der Source-Distribution mit dem Python-Interpreter ausgeführt werden. Zum Ändern wird nur ein Texteditor benötigt, beim nächsten Start des ECMan sind die Änderungen schon aktiv.

Erforderliche Pakete installieren

  • Python > 3.7, inklusive PIP
  • ins ECMan-Wurzelverzeichnis wechseln, im Terminal folgenden Befehl ausführen
pip install -r requirements.txt

ECMan starten

im Terminal ins ECMan-Wurzelverzeichnis wechseln, dann

python3 ECMan.py

Binärdistribution erstellen

Anhand dieser Anleitung wird auf Windows-Systemen aus den Qython-Quellen eine ausführbare EXE für Windows generiert.

CLI-basiert

  1. Zusätzlich zu den o.g. pip-Packeten wird der pyinstaller benötigt:

    pip install pyinstaller==3.4 (??)

  2. im Terminal ins Wurzelverzeichnis vom ECMan wechseln, dann folgendes Kommando ausführen:

    pyinstaller -i images\green_orca.ico --add-data .\scripts;.\scripts ECMan.py

  3. das Kompilat befindet sich im Unterordner dist und kann auf beliebige WIN10-PCs portiert werden.

Mit GUI: auto-pi-to-exe

  1. Zusätzlich zu den o.g. pip-Packeten wird der pyinstaller benötigt:

    pip install auto-pi-to-exe
