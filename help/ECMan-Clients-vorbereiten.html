Clients für die ECMan-Deploymentsoftware müssen vor erstmaliger Benutzung das ConfigureRemoteForAnsible.ps1 Skript ausführen.

Dieses Skript erledigt folgende Aufgaben

  1. Windows Remotemanagementservice aktivieren
  2. Ports für den Windows Remotemanagementservice freigeben
  3. administrativen EC-Man Benutzer angelegen
  4. Benutzer "student" anlegen
  5. imagemagick installieren
  6. Zertifikate für verschlüsselte Datenübertragung erstellen

Namen und Passwörter für o.g. Benutzer sollten vor dem Einspielen des Skripts im Skript selbst angepasst werden.
